<p>Les prix médians* sont établis par les notaires de France sur les transactions réalisées. Voici un comparatif quartier par quartier des prix au m2 de l&rsquo;<a rel="noreferrer noopener" aria-label="immobilier ancien dans le 13001 (opens in a new tab)" href="https://fr.wikipedia.org/wiki/1er_arrondissement_de_Marseille" target="_blank">immobilier ancien dans le 13001</a> en 2018.</p>



<p>Le nombre de vente réalisées dans le 13001 pour l&rsquo;année 2018 : 321 pour un prix médian de 2 266€ le m2. En ce qui concerne le prix au m2 de l&rsquo;immobilier ancien dans le 13001, c&rsquo;est dans le quartier de Thiers que le prix médian est le plus élevé : 2 272€ le m2 contre seulement 1 870€ dans le quartier de Belsunce. </p>



<h2 class="wp-block-heading">Vision globale du 1er arrondissement</h2>



<p>C&rsquo;est clairement le <a href="/nous-les-syndics-marseillais-soyons-a-la-hauteur/">quartier le plus fragile </a> de ces derniers mois, le drame de la rue d&rsquo;Aubagne nous a frappé de plein fouet. Les mois à venir pour la reconstruction et les transactions vont être compliqués. Les prix vont probablement baisser compte tenu de la réputation d&rsquo;insalubrité et des travaux à engager dans les prochaines années. </p>



<p>Le 1er arrondissement reste le quartier ambivalent de la ville, mixant les inégalités sociales avec les merveilles d&rsquo;architecture. Il n&rsquo;y a quasiment que de l&rsquo;immobilier ancien.</p>
