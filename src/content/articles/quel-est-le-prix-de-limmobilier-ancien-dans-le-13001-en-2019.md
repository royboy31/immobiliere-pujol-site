<p>Les prix médians* sont établis par les notaires de France sur les transactions réalisées. Voici un comparatif quartier par quartier des prix au m2 de l'<a href="https://fr.wikipedia.org/wiki/1er_arrondissement_de_Marseille" target="_blank" rel="noreferrer noopener" aria-label="immobilier ancien dans le 13001 (opens in a new tab)">immobilier ancien dans le 13001</a> en 2019.</p>
<p>&nbsp;</p>
<p>Le nombre de vente réalisées dans le 13001 pour l'année 2019 : 278 pour un prix médian de 2 260€ le m2. En ce qui concerne le prix au m2 de l'immobilier ancien dans le 13001, c'est dans le quartier de l'Opéra que le prix médian est le plus élevé : 2 640€ le m2 contre seulement 1 970€ dans le quartier de Belsunce.</p>
<p>&nbsp;</p>
<h2>Vision globale du 1er arrondissement</h2>
<p>&nbsp;</p>
<p>C'est clairement le <a href="https://www.immobiliere-pujol.fr/nous-les-syndics-marseillais-soyons-a-la-hauteur/">quartier le plus fragile </a> de ces derniers mois, le drame de la rue d'Aubagne nous a frappé de plein fouet. Les mois à venir pour la reconstruction et les transactions vont être compliqués. Les prix vont probablement baisser compte tenu de la réputation d'insalubrité et des travaux à engager dans les prochaines années.</p>
<p>&nbsp;</p>
<p>Le 1er arrondissement reste le quartier ambivalent de la ville, mixant les inégalités sociales avec les merveilles d'architecture. Il n'y a quasiment que de l'immobilier ancien.</p>
